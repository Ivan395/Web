Tecnicas de pruebas de software

Su objetivo es proporcionar información objetiva e independiente sobre la calidad del producto a la parte interesada o stakeholder. Es una actividad más en el proceso de control de calidad.

Herramientas CASE

Son diversas aplicaciones informáticas o programas informáticos destinadas a aumentar la productividad en el desarrollo de software reduciendo el costo de las mismas en términos de tiempo y de dinero.

Gestión de calidad del software

La calidad del software se define como la adecuación a los requisitos de funcionalidad y rendimiento explícitamente expresados, a los estándares de desarrollo explícitamente documentados y a las características implícitas que se esperan del software profesional.
